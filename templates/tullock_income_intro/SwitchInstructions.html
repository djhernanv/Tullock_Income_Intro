In the following series of sequences that each lasts {{ Constants.time_in_minutes|apnumber }} minutes you can earn money. For each sequence in which you count the correct number of "a"s you will receive {{ Constants.tokensper_string }} Token (Tk). That is your wage. At the end of this session, one Token will be converted into {{ Constants.eurosper_token }} Euro. Every participant has to count the same letter ( a ) in the same sequences of letters that are displayed in the same order such that everybody faces the same difficulty in counting the letters.

You will start solving short sequences. After each sequence you solve, the length will increase by {{ Constants.increase_per_string|apnumber }} characters.

At any time, you have the option to change to the switch mode. In this mode, you will earn 1 Token per {{ Constants.secondsper_token|apnumber }} seconds without solving any tasks. For instance, spending 13 seconds in the switch mode earns you 1.3 Tk. Since the difficulty of the task increases, at a certain point you will earn more from staying in the switch mode than from solving tasks. Think therefore about when it is best for you change to the switch mode.

Important: Once you are in the switch mode you cannot go back to the letter counting task!

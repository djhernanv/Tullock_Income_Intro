In the following rounds of counting letters you can earn money. For each sequence in which you count the correct number of "a"s you will receive {{ Constants.tokensper_string }} Token (Tk). That is your wage. At the end of this session, one Token will be converted into {{ Constants.eurosper_token }} Euro.

You will start solving short sequences. After each sequence you solve, the length of the sequences will increase by {{ Constants.increase_per_string|apnumber }} characters. The sequences presented and their order are identical for all participants. Also, all participants have to count the same letter ( a ) such that everybody faces the same difficulty in solving the task.

At any time, you have the option to change to the switch mode. In this mode, you will earn 1 Token per {{ Constants.secondsper_token|apnumber }} seconds without solving any tasks. For instance, spending 13 seconds in the switch mode earns you 1.3 Tk. Since the difficulty of the task increases, at a certain point you will earn more from staying in the switch mode than from solving tasks. Think therefore about when it is best for you change to the switch mode.

Each round lasts {{ Constants.time_in_minutes|floatformat:0 }} minutes ({{ Constants.t }} seconds) in which you can either solve tasks or spend time in "switch" mode. A timer in yellow above the sequence will show you the time left in the current round and a green counter will tell you how many sequences you have solved so far.


Important: Once you are in the switch mode you cannot go back to the letter counting task!
